# Metrology Teaching WebApp

Web front end for [Metrology_teaching](https://github.com/SimonTett/Meteorology_teaching)

## Setup with `uv`
```bash
uv venv
. .venv/bin/activate
uv pip compile requirements.in -o requirements.txt
uv pip install -r requirements.txt
```

## Running

```bash
. .nvenv/bin/activate
./run_gunicorn.sh
```
